## 📖 使用说明

## 💻 支持的平台

- **macOS**: 10.15 (Catalina) 或更高版本
- **Windows**: Windows 10/11 (64位)

## ✨ 主要功能

- ✅ 支持 **BitBrowser** 和 **Hubstudio** 指纹浏览器
- ✅ 批量举报用户和商品
- ✅ 多线程并发执行
- ✅ 举报历史记录和数据持久化
- ✅ 白名单管理
- ✅ 智能防重复举报
